# busSeat

첫번째 링크드 리스트 프로젝트.

버스를 관리하고 좌석현황을 확인할 수 있는 관리자와, 버스 좌석을 예매하고 교체, 취소할 수 있는 사용자가 있다.

좌석 예약을 통하지 않고 좌석교체로 바로 들어가게 되면 해당 좌석에서 교체나 취소가 불가능하니 시간나면 수정 예정.
